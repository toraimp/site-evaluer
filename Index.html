Les différents langages de programmation


  • L'Assembleur (ASM)
    • Un langage d'assemblage ou langage assembleur est, en programmation informatique, le langage de plus bas niveau qui représente le langage machine sous une forme lisible par un humain. Les combinaisons de bits du langage machine sont représentées par des symboles dits « mnémoniques » (du grec mnêmonikos, relatif à la mémoire), c'est-à-dire faciles à retenir. Le programme assembleur convertit ces mnémoniques en langage machine, ainsi que les valeurs (écrites en décimal) en binaire et les libellés d'emplacements en adresses, en vue de créer par exemple un fichier objet ou un fichier exécutable.

  • Le COBOL
    • Cobol est un langage de programmation créé en 1959 (officiellement le 18 septembre 1959)2. Son nom est l’acronyme de COmmon Business Oriented Language qui révèle sa vocation originelle : être un langage commun pour la programmation d'applications de gestion. Aujourd'hui, il est surtout utilisé dans les secteurs de la banque, des assurances, des grandes administrations

  • Le BASIC
    • En programmation, le BASIC (acronyme pour Beginner's All-purpose Symbolic Instruction Code, littéralement « Code d'instruction symbolique multi-usages du débutant »), est une famille de langages de programmation de haut niveau ayant pour but la facilité d'utilisation.

  • Le javascript
    • Framework : React, Angular, Vue
    JavaScript est un langage de programmation qui permet d’implémenter des mécanismes complexes sur une page web.

  • Le C et le C++
    • Framework : ULFIUS
    Le langage C++ est un des langages les plus célèbres au monde. Très utilisé, notamment dans le secteur des jeux vidéo qui apprécie ses performances et ses possibilités, le C++ est désormais incontournable pour les développeurs.

  • Le Visual Basic
    • Framework : .NET
    Visual Basic (VB) est un langage de programmation événementielle de troisième génération ainsi qu'un environnement de développement intégré, créé par Microsoft pour son modèle de programmation COM.

  • Le Delphi
    • Framework : FrameworkAOP
    Delphi est à la fois un langage de programmation orienté objet et un environnement de développement intégré pour ce langage. L'EDI Delphi est un EDI propriétaire fonctionnant sous Windows créé en 1995 par Borland.

  • HTML
    • Framework : Bootstrap, Semantic UI, Clank, Skeleton, etc...
    L'HyperText Markup Language, généralement abrégé HTML, est le langage de balisage conçu pour représenter les pages web.

  • Le Flash
    • Framework : ActionScript
    Il permet la manipulation de graphiques vectoriels, d'images matricielles et de scripts ActionScript en vue de créer des contenus multimédia (animations, vidéos, jeux, applications...) destinés à être publiés sur Internet.

  • PHP
    • Framework : CodeIgniter, Agavi, YAF etc...
    PHP (officiellement, ce sigle est un acronyme récursif pour PHP Hypertext Preprocessor) est un langage de scripts généraliste et Open Source, spécialement conçu pour le développement d'applications web. Il peut être intégré facilement au HTML.





    Les différents IDE, leurs particularités et objectifs


  • Visual Studio Community C#, Basic, HTML 5. Devenu gratuit en 3014. Pour Windows.

  • Eclipse en Java, plateforme de développement multi-langages intégrant les mêmes outils. Vous pouvez ajouter vos propres outils !

    • C'est un logiciel difficile à appréhender et peu apprécié.
    On se plaint notamment des plugins instables, la lenteur, et la difficulté à accéder aux composants.
    Ses avantages sont une bonne gestion de SVN et CVS, une large gamme de langages supportés.

  • KDevelop.

    • EDI dédié principalement aux applications en C++ mais qui supporte aussi d'autres langages dont OpenCL.
    Pour Linux et Windows.


  • IntelliJ IDEA.

    • La Community Edition est gratuite.
    Il se distingue par son fond gris par défaut, réalise des applications Java et HTML 5 pour le bureau ou pour Android en version CE, et d'autres langages pour la version pro.
    Bon debogueur, support pour le refactoring.
    Pour Windows/Mac/Linux.
